### 🟢 DESCRIPTION
**ICDK-LEGACY** — это высокотехнологичный Internal-модуль, спроектированный для глубокой манипуляции игровым окружением. Система использует современные методы перехвата функций и парсинга схем данных в реальном времени.

### 🛠 TECHNICAL FEATURES
*   **VMT Hooking:** Бесшовный перехват виртуальных таблиц.
*   **Manual Mapping:** Скрытая инъекция модуля в память процесса.
*   **FrameStageNotify Sync:** Синхронизация рендера без задержек.
*   **Schema System:** Автоматический парсинг оффсетов из client.dll.

### 🚀 INSTALLATION
1. Запустите `ICDK_Injector.exe`.
2. Выберите процесс `cs2.exe`.
3. Нажмите `INJECT_INTERNAL_CORE`.
4. Используйте веб-панель для управления.
